alpha release: limited functionality

the following limitations are in place while SOTU-db is in pre-release:
  • text search only accepts single words
  • queries that return 0 or 1 results may display incorrectly
  • SOTUs available for text search: 1978 - 2017
  • sentiment for text search is always assessed by sentence, chunkSize cannot be selected
  • years available for by-year sentiment analysis: 1981, 1985, 1989 - 1993, 1997, 2001, 2005, 2009, 2013, 2017
  • chunkSize sets the size of chunks of text (in words) for which SOTU-db will return sentiment. Smaller chunkSize means more datapoints, less context.
  • SOTU-db can currently only handle one session at a time. If another use attempts to access SOTU-db while you are using it, you will experience strange behaviors
  • you may encounter other limitations or unexpected behaviors, or the site may go offline entirely without warning
